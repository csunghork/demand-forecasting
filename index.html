📘 Model Reference
Validation Logic:
• Uses rolling forward walk-forward validation
• Start train = first (N − V) periods
• Forecast 1 step ahead, compare to next actual
• Append actual to train, move forward, repeat V times

Included Models:
• Naive, Seasonal Naive, SMA, Drift
• ADIDA, IMAPA, Croston/SBA/TSB
• ETS Auto, Holt-Winters, SES, DES
• ARIMA, SARIMA, Theta
• Prophet-style, XGBoost-style

Winner Ranking:
• Uses composite rank score
• Score combines MAPE, NMAE, MAE%, sMAPE, bias
• Intermittent demand adds sparse-demand penalties

Outlier Handling:
• Rolling median + MAD clipping on training only

Input Conversion:
• PostingDate → Year, Month, Week(1-4)
• Week bucket = days 1-7, 8-14, 15-21, 22-end

Notes:
• Prophet-style = pure JS changepoint + Fourier seasonality
• XGBoost-style = pure JS gradient-boosted tree stumps
